Мини-справочник по дополнительным тегам с примерами

Итак, есть множество тегов, которые как таковые мы не будем изучать в нашем курсе, но которые знать не помешает.

Для удобства изучения разобъем их на несколько групп:

1. Форматирование текста

1.1. Теги <dl><dt> и <dd> для создания списка определений

Пример Результат примера 
<dl>
    <dt>Какое-нибудь одно определение</dt>
        <dd>Подробное и, возможно, очень длинное 
        описание этого определения</dd>
    <dt>Какое-нибудь другое определение</dt>
        <dd>Такое же подробное и не менее длинное 
        описание этого определения</dd>
</dl>
Какое-нибудь одно определение
Подробное и, возможно, очень длинное описание этого определения
Какое-нибудь другое определение
Такое же подробное и не менее длинное описание этого определения

1.2. Тег <ins> для выделения добавленного текста

В этот текст добавлено <ins>новое словосочетание</ins>, 
обратите внимание!

1.3. Тег <del> для выделения удаленного текста

А из этого текста удалено <del>старое словосочетание</del>, 
стало лучше!

1.4. Тег <sub> для отображения текста в нижнем индексе

Используйте этот тег для написания, 
например, формулы воды: H<sub>2</sub>O

1.5. Тег <sup> для отображения текста в верхнем индексе

А этот тег - для записи квадратного уравнения: 
ax<sup>2</sup>+bx+c=0
 А этот тег - для записи квадратного уравнения: ax2+bx+c=0

горизонтальная полоса прокрутки.
