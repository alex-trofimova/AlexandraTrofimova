Тег <picture>

Тег <picture> создает возможность гибко определять ресурсы изображения, а именно:
при наличии нескольких источников будет использоваться первый, который соответствует предпочтениям.

Элемент <picture > содержит два разных тега: один или несколько тегов <source> и один <img>.

Информация

Атрибуты тега <source>, обязательные для реализации эффекта
Атрибут Описание Пример
srcset определяет URL-адрес изображения для отображения srcset="img1.png"
media принимает любой допустимый запрос (указывается в CSS) media="(min-width: 400px)"

Пример

Код:

<picture>
    <source media="(max-width: 1000px)" srcset="img/right.jpg">               
    <source media="(max-width: 1200px)" srcset="img/front.png">
    <img src="img/left.png" alt="Cars" style="width: auto;">
</picture>
